# MyCodingAssistant - Shared Package

Shared types and utilities used across the MyCodingAssistant monorepo.

## Contents

- TypeScript type definitions for messages, services, sessions, and models
- WebSocket event types for client-server communication
- Service configuration types
- Theme types and color definitions

## Usage

```typescript
import { Message, ServiceStatus, SessionInfo } from '@mca/shared';
```

## Building

```bash
npm run build
```

This compiles TypeScript to JavaScript with type declarations.
